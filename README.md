 #My Portfolio Website

This is a personal portfolio website built using HTML, CSS, and Bootstrap.  
It showcases my skills, projects, certifications, and education as a beginner web developer.

## Technologies Used

- HTML
- CSS
- Bootstrap 5

## Website Sections

- Home
- Services
- Projects
- Contact
- Certifications
- Education

## How to Run

1. Download or clone the project files  
2. Open `index.html` in a web browser  

No installation or setup required.

## Notes

- The contact form is for display only
- Content and images can be customized
- Responsive design for mobile and desktop
